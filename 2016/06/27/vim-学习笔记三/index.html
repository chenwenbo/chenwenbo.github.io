vim_学习笔记三

2016-06-27 |
  • . : 执行上一次操作
  • * : 查找下一个光标所在的单词
  • # : 查找上一个光标所在的单词
  • 3i+单词+esc -> 重复三次输入单词
  • vim + 文件名 ->直接创建该文件
  • ge : 效果同 b
  • :s/oldworld/newworld/ ->替换当前行第一个匹配
  • :s/oldworld/newworld/g ->替换当前行所有匹配
  • :%s/oldworld/newworld/ ->替换所有行的第一个匹配
  • :%s/oldworld/newworld/g ->替换所有行的所有匹配
  • ^ ->移动到本行第一个非空字符
  • 0 ->移动到本行第一个字符
  • :s 替换
  • :s% 全局替换
  • /g 替换所有
  • ：split 分屏
  • ：bn切换到上一个文件
  • ：bp切换到下一个
  • 文件
  • ctrl+ww 切换文件
  • ctrl+w 分屏操作功能键
  • :only 仅保留一个屏
  • ctrl +o 跳转到上一次编辑的位置
  • ctrl+i 跳转到下一次编辑的位置
